Base de datos: Firebase

El presente proyecto consiste en el desarrollo de una aplicación móvil de citas dirigida a los estudiantes de la Universidad Unidades Tecnológicas de Santander (UTS),
inspirada en el modelo de interacción de aplicaciones como Tinder. 
La aplicación será desarrollada utilizando el framework Flutter y el lenguaje de programación Dart, permitiendo la generación de una aplicación multiplataforma con 
posibilidad de despliegue en formato APK.
El sistema permitirá a los estudiantes conectar con otros compañeros que compartan similitudes académicas, como carrera y semestre, así como intereses personales en común. La aplicación incorporará un sistema de “match” que habilitará un chat privado entre usuarios cuando exista interés mutuo. Además, se implementarán mecanismos de protección de datos personales y autenticación segura para garantizar la privacidad y seguridad de los usuarios.
Para la gestión y almacenamiento de la información se utilizará Firebase como plataforma backend, aprovechando sus servicios de base de datos en tiempo real, autenticación y almacenamiento en la nube. El objetivo principal es ofrecer una herramienta digital intuitiva, segura y fácil de usar que fomente la interacción social dentro del entorno universitario.
